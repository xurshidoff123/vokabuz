## ✨ Xususiyatlar

- 🔐 **Login sahifasi** — email va parol bilan kirish
- 🌗 **Dark / Light mode** — avtomatik mavzu almashish
- 🌍 **Ko'p tilli interfeys** — O'zbek va Ingliz tillari
- 📊 **Dashboard** — XP ball, streak, kunlik maqsad statistikasi
- 📖 **Darajalar** — A1, A2, B1, B2, C1, C2 (CEFR standartlari)
- 📝 **Darslar** — strukturaviy o'quv materiallari
- 🃏 **Flashcardlar** — so'z yodlash kartalari (flip animatsiyasi bilan)
- 🧠 **Quiz** — bilimlarni tekshirish testi
- 📜 **Sertifikatlar** — darajani tasdiqlovchi hujjatlar
- 👑 **Premium** — to'lov sahifasi bilan darajalarni ochish
- 🔔 **Bildirishnomalar** — toast notification tizimi
